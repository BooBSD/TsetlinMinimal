Minimal Tsetlin Machine implementation in just **200** lines of code.

This may be useful for implementation in other programming languages.

## How to run MNIST example

First, install the [Julia language](https://julialang.org) by running the following command and following the installation instructions:

```shell
curl -fsSL https://install.julialang.org | sh
```

Run the following command:

```shell
julia -t auto mnist.jl
```

> [!NOTE]  
> A faster and more optimized version is available here:  
> 🚀 https://github.com/BooBSD/Tsetlin.jl
